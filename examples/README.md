# examples — демонстрационные песочницы

Примеры разнесены по режимам. В каждой секции — свой `README.md` с инструкцией
запуска и таблицей ожидаемых результатов.

- [`normalizer/`](normalizer/) — дерево фикстур для нормализатора (`fs-normalizer`):
  по подпапке на правило, плюс кейсы конвейера, фильтра `.fs-ignore` и крайних
  сценариев. Нормализатор **меняет** фикстуры на месте, поэтому рядом лежат скрипты
  отката `reset.sh`/`reset.command`/`reset.bat` (возвращают дерево к версии из git).
- [`checker/`](checker/) — дерево-песочница для проверки структуры (`fs-checker`)
  с файлом правил `.fs-check`. Проверка структуру **не меняет**, поэтому скриптов
  отката здесь нет.

Запускать примеры удобно прямо из этих каталогов (см. README секции): командами
`fs-normalizer`/`fs-checker` или обёртками из [`bin/`](../bin/).
