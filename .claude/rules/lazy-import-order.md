# Порядок lazy-import (`importlib.import_module`)

> Claude-эквивалент [`.cursor/rules/lazy-import-order.mdc`](../../.cursor/rules/lazy-import-order.mdc). Применяется точечно — при работе с последовательными блоками `importlib.import_module`.

В блоке последовательных загрузок через `importlib.import_module` порядок строк
должен совпадать с порядком первого использования загруженных имён ниже по коду.

## Область применения

- Только конструкции вида `name = importlib.import_module(...).attr`.
- Только последовательные блоки присваиваний внутри одной функции.
- Вне этих блоков правило не применяется.

## Критерий

- Если имя `a` впервые используется раньше `b`, то строка загрузки `a` должна идти
  раньше строки загрузки `b`.
- Одиночная загрузка не проверяется.

## Исключения

- Независимые ветви `if/elif/else`, где порядок использования зависит от ветки.
- Случаи, где имя ни разу не используется в текущем теле функции.

## Цель

- Упростить чтение `run`-функций: сверху видно те зависимости, которые сразу идут в
  работу в том же порядке.
